Bem vindo ao Inventario LEWG

#####################################################################################

O projeto da empresa LEWG realizado pela turma CAI M2 TI, desenvolvido por  6 alunos da Escola SENAI de Informática, tem o intuito de melhorar a vida de um empreendedor com sistema de organização de estoques, cadastro, remoção de patrimônios, manutenção e atualização da gestão de ativos atráves de um site especializado.

#####################################################################################

Siga as instruções para obter êxito na utilização do sistema:

1- O usuário deve ter acesso a internet;
2- O usuário deve estar logado como root/administrador;
3- Execute em seu terminal o comando "git clone https://github.com/evelynalves/PROJETOLEWG.git"
4- Execute em seu terminal o comando "bash install.sh"

#####################################################################################

Funcionalidade do script de instalação

- Instala Apache2
- Configura a2enmod
- Configura UTF-8
- Redireciona todos os arquivos para seus respectivos diretórios.

####################################################################################

Configure o redirecionamento de porta, e acesse pelo navegador o ip configurado.

####################################################################################

Login
 
O sistema possui três níveis de usuários, usuário comum, técnico e administrador.

O usuário comum terá a opção de listar todas os patrimônios de acordo com a localidade, andar, sala, número de patrimônio, número série, modelo ou marca.

O técnico terá as opções de cadastrar e remover patrimônios, além de listar todas as localidades.

O administrador tera a opção de cadastrar e remover usuários, cadastrar e remover patrimônios, além de listar todas as localidades.
